Die ultimative, neue Spiele-APP

Warum diese Webseite?
 Diese Webseite ist ein Projekt für das Fach AEP (Anwendungsentwicklung) in der Berufsschule Freising.

Ziel:
 Ziel dieser Webseite ist eine Memory-Spielseite. Spieler müssen sich Regestrieren und können dann mit einem anderen Spieler Memory spielen. Anzahl der Spiele, sowie die G/V-Rate (Gewinn-/ Verlustrate) ergeben einen Gesamtpuktestand und werden von jedem Spieler im Leaderboard aufgeführt.

Memory ist ein Gedächtnisspiel für 2-10 Spieler.
Es gibt mittlerweile regelmäßige Memory-Turniere.
Seit 1990 findet jährlich die "Deutsche Memory-Meisterschaft statt."

Memory spielen
Die Ravensburger AG ist in Deutschland im Besitz der eingetragenen Wortmarke "Memory".
Die Spielidee selbst geht zuzrück bis ins 12. Jahrhundert.
